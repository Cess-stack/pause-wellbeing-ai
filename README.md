# Pause – A Wellbeing Companion

Pause is a culturally grounded wellbeing companion designed to help people interrupt overwhelm and return to a grounded state.

This project is inspired by Te Whare Tapa Whā and focuses on calm, non-judgemental support rather than optimisation or tracking.

## Tech (Phase One)
- Frontend: React
- AI: Gemini
- Focus: Nervous system regulation, grounding, reflection
